# Real Crypto Alerts - Mobile (Android)

Versão mobile do aplicativo Real Crypto Alerts para Android.

## Instalação

```bash
cd mobile
npm install
```

## Desenvolvimento

```bash
npm start
```

## Build APK

```bash
npm run build:apk
```

## Recursos Móveis

- ✅ Interface otimizada para touchscreen
- ✅ Controle de volume grande e fácil
- ✅ Sem botões de janela (WIN, TOP, MIN, MAX, CLOSE)
- ✅ Cards que pulam para o topo quando emitem alerta
- ✅ Suporte a USD e BRL
- ✅ Alertas de preço com som
- ✅ Gerenciamento de múltiplas moedas
