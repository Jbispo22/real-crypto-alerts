## 📱 Sobre o Aplicativo

O **Real Crypto Alert** é o aplicativo definitivo para quem acompanha o mercado de criptomoedas e não quer perder nenhuma oportunidade de compra ou venda.

### ✨ Funcionalidades Principais

- 🎯 **Alertas sonoros instantâneos** quando o preço atingir sua meta
- 🔁 **Alerta contínuo inteligente** (avisa se você digitar valor errado)
- 💰 **13.000+ criptomoedas** suportadas via CoinGecko
- 📋 **Botão "Colar"** inteligente para links de qualquer exchange
- 🌙 **Funciona em segundo plano** (tela bloqueada)
- 💱 **Conversão USD ↔ BRL** instantânea
- 🎚️ **Controle de volume** ajustável

### 🔗 Exchanges Suportadas

- Mercado Bitcoin
- Binance
- Coinbase
- KuCoin, Gate.io, OKX, Bybit
- CoinGecko, CoinMarketCap
- E qualquer outra via link direto!
